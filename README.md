# CsharpXmlEditor
Small tool that helps editing XML data in a simple table.

<img width="586" height="493" alt="image" src="https://github.com/user-attachments/assets/6d6582a7-6139-49fd-958c-c668c634628c" />

1. Open XML file
2. Add XPath search, for examle // and then the tag name, for example //Field selects all <Field> tags
3. Press search
4. Configure mappings, for example Column "Width" should map value ./Width relative to //Field
5. Press show
6. Edit inline
7. Export which saves new XML file with the changed values from the data grid
